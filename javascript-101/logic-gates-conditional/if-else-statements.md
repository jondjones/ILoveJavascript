# IF & ELSE

IF/ELSE Statment
```javascript
if (true) {
    console.log('true');
} else {
    console.log('false'); 
}
```

IF/ELSE IF/ELSE Statment
```javascript
if (true) {
    console.log('true');
} else if (false) {
    console.log('false'); 
} else {
    console.log('NOt Possible');    
}
```

Examples
```javascript
const temp = 12;
if (temp <= 32) {
    console.log('Its freezing');  
}
```

```javascript
const age = 45;
if (age <= 7) {
    console.log('Is Child');  
} else if (age >= 65) {
    console.log('Is Senior');  
} else {
    console.log('Is Adult');     
}
```
